# Ideas

Base journal is minimalistically decorated so that it is sitll aesthetically-pleasing. If user wants further customization, they can decorate it themselves/make use of templates they create.

- Music features:
    - Be able to play a song on loop/pause it/not play it while writing entry
    - Music can be played outside of entry, like in calendar view/flipping through entries
    - Songs use Spotify API to be able to be played/displayed w/ info like the artist's name, song title, lyrics, controls, progress bar
    - Choosing song for entry is optional; can replace space where song album cover is supposed to be with some cute default image
- Calendar view:
    - Can navigate through entries from there
    - Exports playlist of this month's songs + can select multiple months/days
    - Each square in calendar can be decorated if desired (stickers, images, song album covers, etc.)
    - Empty entries might have default image for minimal decoration so it's not just blank 
- Journal customizations:
    - Journal covers that can also be decorated
    - Templates (customizable and default ones) where you can specify page type (lined, dotted, blank, square) and it's background color/image, fonts, colors, etc.
    - Be able to flip through journal
    - Importing/exporting if using local storage
- Entry customizations:
    - Fonts: handwriting (pencil-like, marker-like, pen-like), typed (typewriter, other handwrite-like typed fonts, script)
    - Templates for individual entries
    - Decorations are movable (can be dragged and rotated for scrapbooking-like purposes): stickers (custom, default), washi tape, uploaded images, sticky notes
    - Formatting: similar to features you'd have for Google Docs
    - Will have time written + title of entry
    - A few trackers (mood, day rating, etc.)
    - Be able to turn off spell-checking if desired
