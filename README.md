# Music + Journal

This web app project combines a journal with music-related features. With daily entries, this journaling website also allows a song, like a song-of-the-day, to be chosen for each entry. This combination of recording/listening to music and journaling helps track the evolution of your music taste and listening journey over time along with your everyday personal thoughts.
